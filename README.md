⏱️ Stopwatch Web Application (Full Stack Project)
📌 Overview

This is a full-stack Stopwatch Web Application built using HTML, CSS, JavaScript, Node.js, Express, and MySQL.
It allows users to start, pause, reset the stopwatch and record lap times which are stored in a database.

The project demonstrates frontend + backend integration + database CRUD operations + REST API + deployment.
